# Quiz

A full-stack web-based quiz application designed to create, manage, and take quizzes efficiently. This project demonstrates a structured frontend-backend architecture.

## About

Quiz is a modular quiz platform with separate **frontend** and **backend (API)** components. Users can:

- Take quizzes in the browser  
- Manage quiz questions and answers  
- Expand functionality to track scores and progress  

This project serves as both a learning tool and a foundation for larger web applications.  

---

## Technologies

- **Frontend:** TypeScript, HTML5, CSS3, Angular, NgRX
- **Backend:** Node.js, Postgre
- **Tools & Utilities:** npm, Git, Docker
